Given the above plot, a truncated normal distribution may be suitable for simulation of data. Note that we’ll be using the Wilcoxon, so the exact distributional patterns are less relevant. Compare some simulated data to the real data.

In the following the value provided for sd will track the initial sd, but following the principles of transformation for linear variables.

\[ \text{If } Y = aX + b \text{, then} \\ \text{Var}(Y) = a^2\text{Var}(X) \] For example, for a 50% reduction in Ki67, the sd for the reduced sample will be √(1 - 0.5)σ. Likewise for a 70% reduction, the sd for simulation will be √(1 - 0.7)σ.

ki0 <- 0.4
n <- 20
sigma <- 0.2

Based on the observed PDX data, start with σ = 0.2, an initial Ki67 value of 0.4, simulating 20 observations per group. Whilst σ = 0.2 is higher than that in the observed data, it should be noted that for a truncated normal, the observed sd will have a bias to be lower than the provided value. By way of example, simulating 20 point 1000 times, returned a median sd of 0.186. Using the sqrt function for the sd of reduced Ki67 data may again be conservative, but for power calculations it is best to err on the side of caution.

To generate a power curve, we need power on the y-axis, n on the x-axis with different starting values for ki0. Importantly, power was estimated using α = 0.017 as the threshold for significance, given we require multiple testing to be taken into account, and we have three comparisons.

power_fun <- function(n1, n2, init, reduction, sd = 0.2, alpha = 0.05/3, n_sim = 1e4){
    p <- replicate(
        n_sim,
        {
            y1 <- rtruncnorm(n1, a = 0, b = 1, mean = init, sd = sd)
            y2 <- rtruncnorm(n2, a = 0, b = 1, mean = (1-reduction)*init, sd = sqrt(1-reduction)*sd)
            wilcox.test(y1, y2)$p.value <= alpha
        }
    )
    mean(p)
}

Assuming an initial Ki67 positivity of 20%, choosing a sample size of 20 control + 40 for both treatment arms (N = 100), we have a power:

  • ≥90% to detect Ki67 reduction of 70% or more (assuming ≥20% initial positivity).
  • 75% to detect Ki67 reduction of 60% or more (assuming ≥20% initial positivity).
  • 50% to detect Ki67 reduction of 50% or more (assuming ≥20% initial positivity).

When comparing between already reduced Ki67 positivity in the two treatment arms, this will provide a power of 70% to detect a reduction of 60% or more between the two treatment groups. This assumes a lower initial positivity (~10%) in the higher of the two treatment arms.

CT Scans

Assuming that dogs from the above treatment arms go forward into the next step of the experiment, we need to assess the probability of relapse, and our ability to detect difference. According to the McEwen paper, the 100 day probability of being cancer-free for dogs with tumour volumes > 40cc is 50% of patients.

If we assume the above sample sizes (i.e. 20 control and 40 in the treatment arm), we can perform a Fisher’s Exact Test, given that this is sampling without replacement. Data was simulated using a binomial distribution, simulating the number of dogs with a relapse for a given probability. Power was then estimated using Fisher’s Exact Test for a difference between the control group and the treatment arm. Data for the treatment arm was simulated using the same process, but with a reduced probability of recurrence.

fisher_fun <- function(p1, n1, n2, improvement, n_sim = 1e3, alpha = 0.05){
    res <- replicate(
        n_sim,
        {
            d1 <- rbinom(1, n1, p1) # The estimated recurrence (which is success in this modelling approach)
            ## The treatment group
            p2 <- (1 - improvement)*p1
            d2 <- rbinom(1, n2, p2)
            c(d1, n1-d1) %>%
                rbind(c(d2, n2 - d2)) %>%
                set_colnames(c("relapse", "survival")) %>%
                set_rownames(c("control", "treat")) %>%
                fisher.test() %>%
                .[["p.value"]] %>%
                is_less_than(alpha) 
        }
    )
    mean(res)
}

Conclusion

Based on both the above analyses, the sample sizes of ni = 40 for each of the Ki67 treatment arms would give a consistent power of ≥70% to detect a reduction in Ki67 ≥50% in either treatment arm, assuming the initial Ki67 values are ≥20% in the untreated samples. For comparison between treatment arms, this gives a power of 70% to detect a difference ≥60% in Ki67, assuming Ki67 percentages are above 10% in the higher of the two treatment arms.

This would allow taking 3/4 of dogs forward into the CT scan section of the study. Using ni = 30 gives a power ≥70% to detect a change in the proportion of dogs showing recurrence to being ≤1/3 of what is observed in the untreated group. This can also be explained as a 66% reduction in the recurrence rate.

Increasing this to ni=40 for each CT scan group would return a power of ≥70% to detect a reduced recurrence to be 43% of what is observed in the control group (i.e. a reduction of 57% in the recurrence rate).
