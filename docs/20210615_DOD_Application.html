Data was provided as above with most files being derived from Hickey et al, 2021. File names reflect the figures and data files from this paper.

The models included were E418, HCI005, Gar15-13, UCD4, UCD65 and MH18 (not from the specified paper). Only Vehicle, SARM and P4 treated samples were of interest.

Assessment of Growth Rates

Each individual model will have a variable growth rate and these were first explored in order to gain an understanding of any associated variability.

E418

e418 <- here::here(
  "data/Tumour_power_calculation_data/", 
  "3b - E418 UCD4 E SARM DHT final data.pzfx"
  ) %>%
  read_pzfx(table = "Final growth data") %>%
  dplyr::rename(day = Var.1) %>%
  pivot_longer(
    cols = -day,
    names_to = "tumour",
    values_to = "volume"
  ) %>%
  dplyr::filter(!is.na(volume), volume > 0) %>%
  separate(
    tumour,
    into = c("treatment", "replicate"),
    sep = "_", 
    remove = FALSE
  ) %>%
  mutate(
    treatment = as.factor(treatment) %>% relevel(ref = "E2+PBS"),

  group_by(tumour) %>%

  ungroup()

As such, a variability in the growth rate was chosen to be 0.025 as this represented the 90th percentile of the non-UCD65 (ED9c) experiments, and if including this more variable experiment, this still is approximately equivalent to the 75th percentile. This will also be a conservative over-estimate of the variability for the SARM-treated and highly responding lines.

For initial tumour volume, a value of 8 with inter-replicate variability of 0.4 will be appropriately conservative for those using a Vehicle control. However, for those using an E2 control, an initial volume of 3 with variability of 1.2 will again provide a suitably conservative value.

For overall ‘noise’, a value of 0.25 will be representative for control samples, whilst this should be raised to 0.4 for treated tumours

Power Curves

library(lmerTest)
simulateData <- function(n_control = 8, n_treat = 8, mn0 = 8, sd0 = 0.4, gr_control = 0.08, gr_sd = 0.025, gr_reduction = 0.3, sigma_control = 0.25, sigma_treat = 0.4, ...){
  
  ## Simulate the intercepts
  int_cont <- rnorm(n_control, mn0, sd0)
  int_treat <- rnorm(n_treat, mn0, sd0)
  
  ## Simulate the slopes
  slope_cont <- rnorm(n_control, gr_control, gr_sd)
  slope_treat <- rnorm(n_treat, (1-gr_reduction)*gr_control, gr_sd)
  
  ## Assume measurements every 7 days, for 8 weeks: 56 days
  days <- seq(0, 60, by = 7)
  
  tibble(
    replicate = c(seq_len(n_control), seq_len(n_treat)),
    intercept = c(int_cont, int_treat),
    slope = c(slope_cont, slope_treat),
    sd = c(rep(sigma_control, n_control), rep(sigma_treat, n_treat)),
    treatment = c(rep("control", n_control), rep("treated", n_treat)),
    sample = paste(treatment, replicate, sep = "_"),
    days = lapply(seq_len(n_control + n_treat), function(x){days})
  ) %>%
    unnest(days) %>%
    mutate(log2_volume = rnorm(nrow(.), intercept + slope*days, sd))
  
}
alpha <- 0.01

In all calculations below a significance threshold of α = 0.01 was used. This allows for multiple testing for up to 5 experiments using Bonferroni’s adjustment. Any increase in the number of experiments will require a more stringent threshold again.

For simplicity, the same number of replicates were assumed to be performed in both treated and control arms of the experiments.

Fixed values for all simulations were defined as:
